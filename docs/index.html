Use Image to Mosaic in Your App

        
      

      

        

          
Installation

          
npm install image-to-mosaic

        


        

          
Basic Usage

          
import { Mosaic } from 'image-to-mosaic';

const mosaic = new Mosaic({
  target: '#container',
  image: 'path/to/image.jpg',
  pattern: 'grid',
  density: 4,
  animation: {
    type: 'fade',
    duration: 500,
  },
});

mosaic.render();

        


        

          
Available Options

          
    
            
  • pattern: grid, brick, diamond, strips, voronoi, puzzle
    • 
            
  • animation: fade, scale, flip, slide, scatter
    • 
            
  • delay: sequential, random, center, spiral
    • 
            
  • hover: lift, glow, zoom, tilt, flip, blur, pop, spotlight
